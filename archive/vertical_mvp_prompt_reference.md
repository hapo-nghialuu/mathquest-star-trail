# Vertical MVP Prompt Reference

The reference video shows a prompt similar to:

> Lets create a polish version for the first Vertical MVP slice of this
> application. By polish i mean the assets, animation, business logic and
> functionalities must fully completed. We can generate the assets with
> gpt-image-2 skill and leverage the gsap skill. The design guidance is here
> @json/ and the references are @assets/references/ We must stay aligned with
> our art design.

This package implements that intent as a clean source-of-truth bundle. The
visual assets are not copied from the video; they are generated from original
SVG source in this package.
